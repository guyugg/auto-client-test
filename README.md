# 遊戲桌台控端自動化測試控制台 🎮

本專案是一個用於自動化發送各類遊戲桌台控端指令（包括下注、發牌、宣告結果、確認結算等）的測試工具。為了方便**非技術人員 / 不會程式碼的使用者**執行測試，我們已將專案實作並打包為基於 **React** 的純前端控制面板，支援**隨處開啟即用**！

---

## 🌟 核心特色

1. **純前端直連**：無需啟動本地後端服務，直接使用原生 WebSocket 進行測試連線。
2. **零配置部署**：只需將編譯後的 `dist` 資料夾放上機器，或直接雙擊點開 `dist/index.html`，即可在任何瀏覽器中直接執行測試！
3. **多遊戲支援**：支援 **百家樂 (DC)**、**龍虎 (DT)**、**輪盤 (RO)**、**色碟 (SD)**、**骰寶 (SI)** 五種遊戲桌台。
4. **內建預設值**：各遊戲皆內建常用的 WebSocket URL 與荷官參數，一鍵載入即可直接執行。
5. **即時日誌主機 (Live Console)**：將測試指令與 WebSocket 的收發狀況，透過不同顏色與格式，即時在網頁中動態滾動呈現。
6. **桌台互動畫面 (Landscape)**：提供內嵌式「桌台互動畫面」，可於網頁右側以最適合操作的橫向比例載入荷官與發牌互動網頁，方便一邊測試一邊確認牌局。
7. **智慧參數拼接**：在 UI 修改登入帳密後，系統會自動在建立連線時動態拼裝至 WebSocket URL 中，免除手動修改網址字串的麻煩。

---

## 🚀 如何部署與使用？

### 1. 本地直接使用 (最簡單)
* 直接進入 `dist/` 目錄，雙擊打開 **`index.html`** 即可直接開始進行測試！

### 2. 部署至機器
* 將 **`dist/`** 資料夾中的所有內容上傳到您的靜態網頁伺服器（例如 Nginx、Apache 或隨意一台 Web 伺服器）的指定目錄中即可。
* 由於本專案為 100% 純前端 WebSocket 驅動，因此**部署到任何靜態網站服務，都不會產生 any CORS 跨域阻擋限制**，完全免安裝瀏覽器跨域外掛！

---

## 💻 前端開發與編譯 (React & Webpack)

如果您需要修改 UI 或是調整測試狀態機逻辑，可以在根目錄下進行以下操作：

1. **安裝依賴套件**：
   ```bash
   npm install
   ```
2. **啟動本地開發熱更新 (Hot Reload)**：
   ```bash
   npm run start
   ```
   *這會啟動 Webpack Dev Server 並自動開啟 `http://localhost:3000`*
3. **編譯打包生產版本**：
   ```bash
   npm run build
   ```
   *最新編譯好的檔案會直接輸出至根目錄的 `dist/` 資料夾下*

## 📂 專案檔案結構

- 📁 **`dist/`**: Webpack 編譯打包後的純前端靜態資源，可直接雙擊使用或上傳機器部署。
- 📁 **`src/`**: React 原始碼目錄。
  - 📝 **`src/App.js`**: React 控制面板主要介面與佈局。
  - 🎨 **`src/App.css`**: 面板樣式與橫向自適應 Grid 排版。
  - 📜 **`src/testRunners.js`**: JavaScript 實作的 5 款遊戲 WebSocket 狀態機自動化測試主程序。
- 📁 **`python/`**: 原本的 Python 測試腳本存檔。
  - 📜 **`dc.py` / `dt.py` / `ro.py` / `sd.py` / `si.py`**: 支援從系統環境變數加載配置。可在終端機單獨運行（例如 `python python/dc.py`）。
- ⚙️ **`webpack.config.js`**: Webpack 打包與配置。
- 📦 **`package.json`**: 專案依賴與運行腳本定義。
